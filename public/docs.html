Syarat Penggunaan

Dengan menggunakan Soraa Deployment, kamu setuju tidak menyalahgunakan layanan untuk tujuan ilegal, menyebarkan malware, atau konten yang melanggar hukum.

Batasan Layanan

Setiap pengguna dibatasi ukuran upload maksimal 50MB per file ZIP. Layanan gratis untuk penggunaan personal dan non-komersial.

Perubahan Ketentuan

Kami berhak mengubah syarat ini kapan saja. Perubahan akan diumumkan melalui channel Telegram @depstore11.

Data yang Dikumpulkan

Kami hanya menyimpan data statistik penggunaan (jumlah deploy dan clone) tanpa informasi identitas pribadi.

File yang Diunggah

File ZIP yang kamu upload hanya diproses sementara dan tidak disimpan di server setelah proses selesai.

Kontak

Pertanyaan soal privasi? Hubungi kami di Telegram: @depstore11

Penggunaan Cookie

Soraa Deployment menggunakan cookie sesi yang bersifat teknis dan esensial agar aplikasi berjalan dengan baik.

Tidak Ada Pelacakan

Kami tidak menggunakan Google Analytics, Facebook Pixel, atau alat pelacakan iklan apapun.
